## Installation and Usage

Install OCaml dependencies and the Dune compiler 
After zip download, cs to root directory and run: 

 brew install opam

 opam init 

 opam install dune

 opam install RayGui Raylib

 opam install RayGui Raylib

reference install.txt for further questions!
